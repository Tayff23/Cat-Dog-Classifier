# Cat-Dog-Classifier

## Project Overview

This project is a simple image classification project created using Google Teachable Machine and Python. The model was trained using two classes (Class 1 and Class 2) with a small set of images.

## Tools Used

- Google Teachable Machine

- Python

- TensorFlow / Keras

- Visual Studio Code

## Project Files

- predict_cat_dog.ipynb – Python notebook used to load the model and test an image.

- keras_model.h5 – The trained model.

- labels.txt – Contains the class labels.

- DogTest.webp – Test image used for prediction.

- Screenshots/ – Contains screenshots of the training process and prediction result.

## How It Works

1. Two images were added for each class in Google Teachable Machine.

2. The model was trained.

3. The trained model was exported.

4. The Python notebook was used to load the model.

5. A new dog image was tested using the exported model, and the prediction result was displayed.

## Result

The model successfully predicted the test image as **Class 2** with a confidence score of approximately **99.89%**.
